# System Requirements

## Minimum Requirements

### Android
- **OS Version:** Android 5.0 (API Level 21) or higher
- **RAM:** 2 GB minimum
- **Storage:** 100 MB free space
- **Screen:** 4.5 inches or larger
- **Internet:** WiFi or mobile data connection

### iOS
- **OS Version:** iOS 12.0 or higher
- **Device:** iPhone 6s or newer
- **RAM:** 2 GB minimum
- **Storage:** 100 MB free space
- **Internet:** WiFi or mobile data connection

### Windows
- **OS:** Windows 10 or Windows 11
- **RAM:** 4 GB minimum
- **Storage:** 500 MB free space
- **Internet:** Internet connection for cloud features

## Recommended Requirements

### Android
- **OS Version:** Android 10 or higher
- **RAM:** 4 GB or more
- **Storage:** 200 MB free space
- **Processor:** Snapdragon 670 or equivalent
- **Display:** 5.5-6.5 inch FHD+ display

### iOS
- **OS Version:** iOS 14 or higher
- **Device:** iPhone 11 or newer
- **RAM:** 4 GB or more
- **Storage:** 200 MB free space
- **Display:** 5.5 inch or larger

### Windows
- **OS:** Windows 11
- **RAM:** 8 GB or more
- **Storage:** 1 GB free space
- **Processor:** Intel Core i5 or equivalent
- **Display:** 1920x1080 or higher

## Network Requirements

- **Internet Connection:** Required for:
  - Initial login
  - Cloud synchronization
  - Real-time updates
  - Report generation
  - Backup and recovery

- **Bandwidth:** Minimum 1 Mbps for optimal performance

- **Offline Support:** Yes (local SQLite database)
  - Data syncs when connection returns
  - Full POS functionality offline

## Browser Requirements (Web Version - Future)

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

## Optional Hardware

### Barcode Scanner
- USB 2D/QR code scanner
- Bluetooth barcode scanner
- Compatible with all platforms

### Fingerprint Scanner
- Built-in fingerprint sensor
- Android: Supported on most modern phones
- iOS: Face ID or Touch ID (iOS 12+)
- Windows: Windows Hello or compatible sensor

### Thermal Printer
- 58mm thermal printer (80mm supported)
- Bluetooth or USB connection
- ESC/POS compatible

### Cash Drawer
- USB or Bluetooth enabled
- Standard RJ-11 connector compatible

## Development Requirements

To build and run from source:

### Required Tools
- Flutter SDK 3.13.0 or higher
- Node.js 16 or higher
- PostgreSQL 12 or higher
- Git 2.25 or higher

### Android Development
- Android SDK (API Level 34)
- Android Studio or VS Code with Flutter extension
- Java Development Kit (JDK) 11 or higher

### iOS Development (Mac Required)
- Xcode 13 or higher
- CocoaPods 1.11 or higher
- macOS 11 or higher

### Windows Development
- Visual Studio 2019 or later (with C++ build tools)
- Or MinGW with GCC 7.0 or higher
