# User Manual

## Getting Started

### Login
1. Launch the Cashier Application
2. Enter your username and password
3. Select your language (English/Arabic)
4. Click Login

## Main Dashboard

### Navigation
- **Home**: Main dashboard and quick actions
- **Sales**: Create and manage sales transactions
- **Inventory**: View and manage stock
- **Reports**: View sales and inventory reports
- **Settings**: User preferences and system settings

## Making a Sale

1. Go to **Sales** section
2. Click **New Sale**
3. Select products using the barcode scanner or search
4. Enter quantity for each product
5. Review the cart
6. Select payment method
7. Process payment
8. Print or email receipt

## Managing Inventory

1. Go to **Inventory** section
2. View current stock levels
3. Add stock: Click **Add Stock** and enter quantity
4. Adjust stock: Click **Adjust** for inventory reconciliation
5. Low stock alerts appear automatically

## Viewing Reports

1. Go to **Reports** section
2. Select report type:
   - Daily Sales
   - Monthly Sales
   - Inventory Status
   - Top Products
3. Filter by date range
4. Export as PDF or Excel

## Offline Mode

- Sales continue working without internet
- Transactions sync automatically when online
- Inventory updates when connection is restored

## Language Settings

1. Go to **Settings**
2. Select **Language**
3. Choose English or العربية
4. App restarts with new language

## Troubleshooting

### Cannot Login
- Check internet connection
- Verify username and password
- Clear app cache

### No Internet
- App continues in offline mode
- Sales are queued for sync
- Data syncs when connection returns

### Barcode Scanner Not Working
- Enable camera permissions
- Clean camera lens
- Ensure good lighting
