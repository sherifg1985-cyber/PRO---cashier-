# PRO---cashier

A comprehensive, cross-platform integrated cashier program supporting Windows, Android, and iOS with full Arabic language support.

## 🌟 Features

- **Multi-Platform Support**: Windows, Android, iOS
- **Multi-Language Support**: English & Arabic (RTL support)
- **Point of Sale (POS)**: Complete cashier functionality
- **Inventory Management**: Real-time inventory tracking
- **Sales Reports**: Comprehensive sales analytics
- **Receipt Management**: Digital & physical receipts
- **Payment Processing**: Multiple payment methods
- **User Authentication**: Role-based access control
- **Offline Mode**: Work without internet connection
- **Cloud Sync**: Automatic synchronization when online

## 🛠️ Technology Stack

- **Frontend**: Flutter (cross-platform UI)
- **Backend**: Node.js with Express
- **Database**: PostgreSQL (server), SQLite (local)
- **Real-time**: WebSocket
- **Authentication**: JWT
- **Cloud Services**: Firebase (optional)

## 🚀 Quick Start

### Prerequisites
- Flutter SDK (v3.0+)
- Node.js (v16+)
- PostgreSQL (optional for development)

### Development Setup

1. Clone the repository
2. Navigate to `flutter_app` and run: `flutter pub get`
3. Navigate to `backend` and run: `npm install`
4. Follow the setup guides in respective directories
